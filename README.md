# E-Commerce
Technology E-Commerce website with Java and Spring

## 💡 Features
- Login/Register
- Add/Edit/Delete Product(Admin)
- Add Products to Cart
- Search Product
- Buy Products

## 👨‍💻 Used technologies
`Java`, `Spring Security`, `Hibernate`, `MySQL`, `HTML`,`CSS`,`Thymeleaf`, `Spring Data`, `Spring MVC`
